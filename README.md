# Early Disease Detection using Voice

## Overview

This project explores the use of artificial intelligence and signal processing techniques to detect potential diseases from human voice recordings.

Voice recordings contain subtle acoustic biomarkers that can indicate neurological and respiratory conditions. This system focuses on extracting meaningful acoustic features from voice signals and applying machine learning models for early disease prediction.

This project was developed as part of my Master's thesis in Artificial Intelligence.

---

## Motivation

Early disease detection can improve treatment outcomes and support timely medical intervention.

This work investigates how machine learning models can use vocal biomarkers for non-invasive and accessible screening.

## Feature Extraction

The system uses acoustic and speech-based features such as:

- MFCC (Mel-Frequency Cepstral Coefficients)
- Spectral centroid
- Pitch
- Jitter
- Shimmer

These features capture important vocal characteristics that can support disease classification.
